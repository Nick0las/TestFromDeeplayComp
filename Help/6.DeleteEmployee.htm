6. Удаление данных о персонале.

Удаление должностной информации доступно только из окна "Информация о сотрудниках".

Для удаления необходимо выбрать сотрудника в правой части окна и кликнуть по кнопке "Удалить должностную информацию"

Удаление персональных данных (анкеты сотрудника) доступно из окна "Информация о сотрудниках" и из главного окна приложения.

Для удаления персональных данных должностные данные должны быть уже удалены из базы данных.

При удалении "Должностной информации" приложение спросит, нужно ли удалить персаналные данные из БД. При отказе, анкетные данные сохранятся.

Анкетные данные позже можно удалить из главного окна приложения.
