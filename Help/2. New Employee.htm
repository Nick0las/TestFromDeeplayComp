1. Создание нового сотрудника.

Для создания нового сотрудника необходимо кликнуть по кнопке "Новый сотрудник" главного окна приложения

После открытия окна создания нового сотрудника необходимо заполнить текстовые поля (анкета сотрудника):

1. Фамилия, имя, отчество (обязательны для заполнения);

2. Дата рождения, домашний адрес, номер телефона (обязательны для заполнения)

Для создания должностного профиля предприятия необходимо заполнить поля, относсящиеся к должности:

1. Должность (обязательно для заполнения);

2. Отдел (ТОЛЬКО для должности "Руководитель отдела (подразделения)";

3. Руководитель (для должности "Рабочий");

4. Дата приема на работу (обязтельное поле для всех должностей);

5. Дата окончания договора (только для должности директор);

После заполнения всех необходимых данных о новом сотруднике необходимо кликнуть по кнопке "Добавить"
