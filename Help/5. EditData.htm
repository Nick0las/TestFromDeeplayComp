5. Редактирование данных о персонале.

Редактирование данных осуществляется из окна "Информация о персонале".

Для редактирования необходимо выбрать сотрудника в правой части окна и кликнуть по кнопке

"Редактирование персональных данных" - для изменения данных анкеты сотрудника

"Редактирование должностной информации" - для изменения должностной инфомации.

Редактирование персональных данных:

После изменения необходимых данных необходимо кликнуть по кнопке изменить или отмена для отмены внесенных изменений

При изменении должностной информации можно "уволить" сотрудника, поставив дату увольнения. При этом вся информация о нем останется в БД.
