Главное окно программы.

Главное окно программы предоставляет пользователю основные функции программы:

1. Создание нового сотрудника

2. Просмотр всех сведений о существующих сотрудниках.

3. Удаление анкеты сотрудника (только при отсутствии привязки анкеты к должности предприятия).

4. Открытие руководства пользователя

5. Просмотр основных данных анкеты пользователя (ФИО, дата рождения, дом. ардес, номер телефона).
